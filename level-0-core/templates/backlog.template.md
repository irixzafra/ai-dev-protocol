# Backlog

> Inbox for ideas that aren't tasks yet.
> An idea lives here until it's analyzed and either moved to WORKBOARD or rejected.
> Keep this file short — if an idea sits here unanalyzed for 2+ weeks, reject it or schedule analysis.

---

## How it works

```
Capture → Analyze (dialogue with agent) → Decision
                                              ↓           ↓
                                          WORKBOARD    Rejected
```

**Priority (MoSCoW):**
- `M` — Must: critical, blocking, required now
- `S` — Should: important, next iteration
- `C` — Could: nice to have, when bandwidth allows
- `W` — Won't: not now, may revisit later

**Status:**
- `[idea]` — captured, not yet analyzed
- `[analyzed]` — pros/cons explored, decision pending
- `[accepted → WORKBOARD]` — moved to task queue
- `[rejected]` — not doing it, with reason
