# Audit Mode — Output Format

When "Audit" is activated, the output is always an **executable task list** in this format:

```markdown
## Audit [Project] — [date]
**Estimated score:** [X/100] · **Critical issues:** [N]

### CRITICAL (blocks launch)
- [ ] [P0] Contrast text-muted vs bg: 2.1:1 — needs >=4.5:1 · File: globals.css:18 · Est: 15min
- [ ] [P0] Contact form has no validation — leads are lost · File: CTAFinal.tsx · Est: 2h

### IMPORTANT (affects conversion)
- [ ] [P1] No focus-visible on nav buttons · WCAG 2.2 SC 2.4.11 · Est: 30min
- [ ] [P1] Heading h3 without parent h2 in Products section · Est: 10min
- [ ] [P1] No alt on logo SVG · Est: 5min

### IMPROVEMENT (elevates quality)
- [ ] [P2] prefers-reduced-motion not implemented · Est: 20min
- [ ] [P2] Light mode missing tokens for hero-bg · Est: 45min
- [ ] [P2] Floating CTA on mobile to avoid scroll loss · Est: 1h

### INFORMATIONAL (does not block)
- [ ] [P3] Inter font in display — replace with distinctive sans
- [ ] [P3] JSON-LD missing LocalBusiness schema

**Total estimated:** Xh Ymin
**Recommended next step:** start with P0 then P1 contrast issues
```

Each item includes: priority, issue description, WCAG reference if applicable, file:line, time estimate.

## Pre-Delivery Checklist

### Responsive
See `responsive-qa.md` for full responsive validation procedure with canonical breakpoints (390, 1024, 1440).

### Dark/Light mode
- [ ] Correct CSS variables in `:root` (light) and `.dark`
- [ ] 0 hardcoded colors outside the token system
- [ ] Smooth transition when switching modes
- [ ] next-themes configured in layout root

### Accessibility
See `ui-foundations.md` for the complete WCAG 2.2 AA checklist and neurodivergence guidelines.

### Performance
- [ ] Only transform + opacity in animations
- [ ] next/image on all images
- [ ] Fonts with preload + display: swap
- [ ] No layout thrashing in animations

### Code
- [ ] TypeScript without `any`
- [ ] ESLint without errors
- [ ] Server Components where possible
- [ ] Server Actions for forms

### AI Ethics
- [ ] If generative AI in the UI -> visible indicator (e.g., "Generated by AI")
- [ ] User can regenerate, edit, or ignore AI suggestions
- [ ] Privacy: inform what data is used for personalization
