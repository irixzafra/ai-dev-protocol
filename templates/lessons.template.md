# LESSONS — Corrections Inbox

> `LESSONS.md` is an **inbox**, not a permanent source of truth.
> Every lesson must be graduated to where it actually gets read.
> A lesson stuck here that nobody reads is worthless.
> See `protocol/SELF_IMPROVEMENT.md` for the full graduation model.

---

## Graduation model

| If the correction is about... | Graduate to |
|---|---|
| Agent behavior | rule in `dev.protocol.md` |
| Tool/workflow insight | update the affected skill |
| Automatable check | entry in `.husky/pre-commit` |
| Multi-agent coordination | section in coordination guide |
| Operational setup | runbook in `docs/` |
| Module-specific knowledge | `[reference]` — stays here |

**Lesson status:**
- `[pending]` — captured, not yet graduated. Requires action.
- `[reference]` — module-specific; no systemic change needed.
- `[graduated → path]` — applied; this entry is traceability only.
