## Intent
Add an "Export CSV" button to the users table in the admin panel to allow exporting user data.

## Scope — what will be built
- A new button component in `app/admin/users/page.tsx` labeled "Export CSV".
- The button will trigger a client-side function to fetch user data and generate a CSV file.
- The CSV will include `id`, `email`, `created_at`, and `last_sign_in_at` fields for all users.
- The generated CSV file will be downloaded automatically with a filename like `users-export-YYYY-MM-DD.csv`.

## Scope — what will NOT be built
- Server-side CSV generation.
- Advanced filtering or selection for CSV export.
- Export of additional user fields beyond the specified ones.
- Any changes to the existing user table display or functionality.

## Files likely affected
- `app/admin/users/page.tsx`

- [x] The CSV file contains data for all users displayed in the table.
- [x] The filename is formatted as `users-export-YYYY-MM-DD.csv`.
